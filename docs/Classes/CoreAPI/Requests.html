    Get all the stores for the specified list of dealers. If no sortOrder is provided, it will be sorted by [.nearest, .businessName] Note: If a locationQuery is provided, the stores will be limited to that location. HOWEVER, if the stores are in a different country to the publication, the resulting stores will be any store filtered by the locationQuery. This is an API issue.
