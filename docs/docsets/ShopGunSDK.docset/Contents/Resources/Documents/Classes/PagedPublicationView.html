  • didEnterForeground()

    Tell the page publication that it is now visible again. (eg. when a view that was placed over the top of this view is removed, and the content becomes visible again). This will restart event collection. You MUST remember to call this if you previously called didEnterBackground, otherwise the PagePublicationView will not function correctly.

  • didEnterBackground()

    Tell the page publication that it is no longer visible. (eg. a view has been placed over the top of the PagedPublicationView, so the content is no longer visible) This will pause event collection, until didEnterForeground is called again.
