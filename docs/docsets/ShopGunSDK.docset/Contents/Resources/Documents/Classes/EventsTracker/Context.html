Context

public struct Context

Undocumented

  • location

    The location information of the app’s user. Once set, this will be sent with all future tracked events.

    • A geohash of the location (this will have an accuracy no-greater than ±20km)
    • The timestamp of when that location info was collected.

    It is up to you to collect this info from the user. See the updateLocation(latitude:longitude:timestamp:) method.

    Declaration

    Swift

    public private(set) var location: (geohash: String, timestamp: Date)?
  • updateLocation(latitude:longitude:timestamp:)

    Updates the location property, using a lat/lng/timestamp to generate the geohash (to an accuracy of ±20km). This geohash will be included in all future tracked events, until clearLocation() is called.

    Note

    It is up to the user of the SDK to decide how this location information is collected. We recommend, however, that only GPS-sourced location data is used.

    Declaration

    Swift

    public mutating func updateLocation(latitude: Double, longitude: Double, timestamp: Date)

    Parameters

    latitude

    The latitide to use when generating the location‘s geohash.
    longitude

    The longitude to use when generating the location‘s geohash.
    timestamp

    The date that the lat/lng pair was generated (eg. when the user was discovered to be at that location)
  • clearLocation()

    After this is called, the location geohash/timestamp will be set to nil and no longer sent with future tracked events.
